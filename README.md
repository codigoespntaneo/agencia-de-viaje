# 🚀 CosmoFamily

## Descripción del proyecto

CosmoFamily es una página web para una agencia ficticia de turismo espacial orientada a familias. El sitio presenta experiencias educativas y de aventura en diferentes destinos del sistema solar, destacando la seguridad, el confort y la exploración espacial.

El objetivo del proyecto ha sido recrear una landing page moderna siguiendo un diseño proporcionado mediante mockups.

---

## Mockups

### Diseño general

El diseño incluye:

- Una sección principal (Hero Section) con una llamada a la acción.
- Una sección de destinos galácticos.
- Un diseño visual futurista enfocado en el turismo espacial familiar.
